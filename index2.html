PDF Preview Test Suite

For private internal testing only

Upload a PDF for this Virtual P.O. by clicking the 'Upload' button for each Document type

  • Only one file can be attached to each document type.
  • You can change the file for a document type as often as desired.
  • An existing file for a document type can be deleted.
  • Custom document types can be added that do not fall under one of the pre-defined types by uploading a file for type 'Other...'.
  • There is no limit to the number of custom types and files that can be added. This allows a workaround to have more than one file associated with a document type: e.g., You can have a 'Packing Slip', and then add a custom type 'Packing Slip' (or 'Packing Slip 2').
